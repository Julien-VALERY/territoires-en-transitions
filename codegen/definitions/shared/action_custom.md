## Action personnalisée
Représente une action personnalisée créée pour une EPCI. Stockée en local à l'adresse `epci/mesure_id/uid`

```yaml
ActionCustom:
    uid:
        type: String
    epci_id:
        type: String
    mesure_id:
        type: String
    name:
        type: String
    description:
        type: String
```
